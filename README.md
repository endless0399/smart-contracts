# Deploy smart-contracts on the Ethereum blockchain

## Getting Started

### Dependencies

* nodejs v16.14.0
* hardhat 2.9.1

### Installation
```
npm install --save-dev hardhat
```

### Executing program

* From project root
```
npx hardhat run scripts/deploy.js
cd frontend
npm install
npm run start
```
